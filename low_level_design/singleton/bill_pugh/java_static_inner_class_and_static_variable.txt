-> In Java, a static variable (also called a class variable) is a variable that belongs to the class itself rather
than any particular instance of the class.

# Key Features of a static variable :
1. Belongs to the class, not to objects
    - All instances of the class share the same copy of the variable.
    - Changing it in one instance affects all others.
2. Memory allocation
    - Stored in the method area (class area) of JVM.
    - Allocated once when the class is loaded.
3. Access
    - Can be accessed using the class name: ClassName.variableName
    - Can also be accessed via an object, but not recommended.
4. Default value
    - Initialized with a default value if not explicitly set (0, null, false, etc.).
5. Use case
    - For values that should be common across all instances, e.g., a counter of objects created.


-> A static inner class in Java is a nested class declared with the static keyword.

# Key properties of static inner class :
1. Belongs to the outer class, not instances
- We don’t need an instance of the outer class to create it.
      Outer.Inner obj = new Outer.Inner();

2. Can access only static members of outer class
- Cannot directly access instance variables or methods of the outer class.

class Outer {
    int x = 10;        // instance variable
    static int y = 20; // static variable

    static class Inner {
        void print() {
            System.out.println(y);  // OK
            System.out.println(x);  // ERROR
        }
    }
}

3. Can have its own members
- Can have its own instance variables and methods independent of the outer class.

4. Can be instantiated independently
- No outer instance is required.

5. Can be used to group related classes
- For logical grouping or hiding from external access.
